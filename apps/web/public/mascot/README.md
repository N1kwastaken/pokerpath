# Mascote (ilustrações)

Solte aqui as ilustrações PNG do mascote (naipe de espadas com carinha,
estilo cartoon — fundo transparente, quadrado, ~512x512). O app usa
automaticamente quando o arquivo existir; se faltar, cai num fallback SVG.

Arquivos esperados (1 por expressão):

- happy.png      — sorrindo, joinha (pose padrão)
- excited.png    — empolgado, bracinhos pra cima, boca aberta
- cheer.png      — comemorando (usado ao concluir fase/mundo)
- teaching.png   — "professor", apontando/explicando (telas de aula)
- wave.png       — acenando (boas-vindas / onboarding)
- sad.png        — desanimado (resposta errada / fase não passou)

Dica de prompt para gerar (qualquer IA de imagem):
"Cute cartoon mascot of a black/charcoal spade card suit with a glossy
body, big friendly cartoon eyes, open happy smile with a small tongue,
little arms with white gloved hands giving a thumbs up, soft studio
lighting, flat vector style, transparent background, centered, square."
Gere uma variação por expressão acima, mantendo o MESMO personagem.
